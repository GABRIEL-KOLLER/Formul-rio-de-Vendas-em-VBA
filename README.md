Formulário de Vendas com VBA

#Tecnologias

- Excel
- VBA
- Macros
- Formulário

Desenvolvi um formulário de Vendas com e Excel e VBA, cada cadastro é inserido na planilha formatado. (Data,
Nota Fisca, Vendedor, ID, Quantidade, Preço, Valor total.)
